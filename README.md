# ai-resume-screening-system# 🤖 AI Resume Screening System

A machine learning-based system that analyzes resumes and predicts whether they match a given job description.

---

## 🚀 Features
- Resume keyword extraction
- TF-IDF vectorization
- Logistic Regression model
- Flask API for predictions

---

## 🛠 Tech Stack
- Python
- Scikit-learn
- Flask
- Pandas

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
